Portage d'une application de monitoring réseau vers iOS

Statut : Réalisé (Stage de fin d’études)

Durant mon stage chez Esia, j’ai porté une application mobile Android de supervision réseau, développée en React Native, vers iOS. Ce projet a impliqué le développement de fonctionnalités natives Swift (widgets, notifications push, gestion des certificats SSL), l’intégration avec l’API PHP existante, ainsi que l’ajout de nouvelles fonctionnalités telles que l’acquittement et la prise en charge des erreurs directement depuis l'application.

Projet intégré dans l'écosystème professionnel d'Esia.

Letterboxd Poster Downloader

Statut : En développement

Une application web React/Flask qui permet d'importer son calendrier Letterboxd au format CSV, puis de télécharger les posters des films préférés de l'utilisateur. Cette application combine le front-end React pour une interface utilisateur dynamique avec un back-end Flask pour le traitement des données et l'interaction avec les APIs de films.
